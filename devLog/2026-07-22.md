# 2026-07-22 개발 로그

## 작업 단계

- 단계: Phase 1 사용자와 인증 안정화
- 상태: 진행
- 범위: 사용자 프로필 확장·조회·수정 세로 기능

## 오늘의 목표

- User 프로필과 계정 상태 필드를 Flyway migration으로 추가한다.
- 기존 인증 API와 JWT 계약을 유지하며 시스템 권한 이름을 코드에서 명확히 한다.
- 인증된 사용자의 프로필 조회·수정 API와 화면을 구현한다.
- 정상·입력 검증·인증 없음 계약을 테스트한다.

## 결정 사항

- 계정 상태는 `ACTIVE`, `SUSPENDED`, `WITHDRAWN`으로 고정한다.
- 프로필 수정에서는 `status`, `systemRole`, 이메일, 아이디를 변경할 수 없다.
- 선택 프로필 값을 빈 문자열로 보내면 `null`로 정규화한다.
- 실제 이미지 업로드는 후속 storage 기능 범위이며 현재는 URL/storage key를 담을 필드와 편집 계약만 제공한다.
